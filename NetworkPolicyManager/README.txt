<file_name.java> Is the file copied from AOSP

<file_name_details.odt> or <file_name_details.odt> contains the documentation of the violations.

The documentation is arranged in the following way

File Path: Where the file located in AOSP.
 
Line No: line number in the file
 
Code Block: The red color line have the exact violation.

AST Block : contains the AST of the particular vulnerable code. The AST is huge in size so pasted the AST of those files where the violation is present.
